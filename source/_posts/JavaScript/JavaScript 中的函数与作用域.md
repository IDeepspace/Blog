## JavaScript 中的函数与作用域

### 什么是函数

引用类型的值（对象）是引用类型的一个实例。引用类型是一种数据结构， 用于将数据和功能组织在一起。



### 函数的定义与调用

```javascript
var person1 = { name: '陈星星' };
var person2 = { name: 'Deepspace' };
var person3 = { name: '陈鑫' };
```
